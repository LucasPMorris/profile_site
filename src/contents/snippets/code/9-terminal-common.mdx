These snippets are incredibly common and very useful when working in the terminal. Some of these will become so familiar that you will not even think about them anymore, others it is just easy some time to cut and paste!

---

<br/>

<span id="ls-list" name="List Directory Contents" data-toc></span>

> List Directory Contents
```bash
ls -la
```
**Function:** Lists all files and directories in the current location with detailed information including hidden files, permissions, owner, size, and modification date.

<br/>

---

<br/>

<span id="change-directory" name="Change Directory" data-toc></span>

> Change Directory
```bash
cd /path/to/directory
```
**Function:** Navigates to the specified directory. Use `cd ..` to go up one level, `cd ~` to go to home directory, or `cd -` to go to the previous directory.

<br/>

---

<br/>

<span id="pwd-directory" name="Show Current Directory" data-toc></span>

> Print Working Directory
```bash
pwd
```
**Function:** Displays the full path of the current directory you're working in. Useful for confirming your location in the file system.

<br/>

---

<br/>

<span id="mkdir" name="Create Directory" data-toc></span>

> Create New Directory
```bash
mkdir new-folder
```
**Function:** Creates a new directory with the specified name. Use `mkdir -p path/to/nested/folder` to create nested directories all at once.

<br/>

---

<br/>

<span id="cp-files" name="Copy" data-toc></span>

> Copy Files or Directories
```bash
cp source.txt destination.txt
```
**Function:** Copies files from source to destination. Use `cp -r source-folder/ destination-folder/` to recursively copy entire directories.

<br/>

---

<br/>

<span id="mv-files" name="Move/Rename Files" data-toc></span>

> Move or Rename Files
```bash
mv oldname.txt newname.txt
```
**Function:** Moves or renames files and directories. Can be used to move files to different locations or simply rename them in place.

<br/>

---

<br/>

<span id="remove-files" name="Remove Files" data-toc></span>

> Remove Files
```bash
rm filename.txt
```
**Function:** Deletes the specified file permanently. Use `rm -r folder/` to delete directories and their contents, or `rm -f` to force deletion without prompts.

<br/>

---

<br/>

<span id="view-file" name="View File" data-toc></span>

> View File Contents
```bash
cat filename.txt
```
**Function:** Displays the entire contents of a file in the terminal. Good for quickly viewing small to medium-sized text files.

<br/>

---

<br/>

<span id="text-search" name="File Text Search" data-toc></span>

> Search for Text in Files
```bash
grep "search term" filename.txt
```
**Function:** Searches for specific text patterns within files. Use `grep -r "search term" .` to search recursively through all files in current directory.

<br/>

---

<br/>

<span id="command-history" name="Command History" data-toc></span>

> Command History
```bash
history
```
**Function:** Shows a list of previously executed commands. Use `!number` to re-run a specific command from history, or `!!` to repeat the last command.

<br/>

---

<br/>

<span id="find" name="Find" data-toc></span>

> Find Files and Directories
```bash
find . -name "*.txt"
```
**Function:** Searches for files and directories by name, type, size, or other criteria. This example finds all .txt files in current directory and subdirectories.

<br/>

---

<br/>

<span id="disk-usage" name="Disk Usage" data-toc></span>

> Check Disk Space Usage
```bash
df -h
```
**Function:** Displays disk space usage for all mounted filesystems in human-readable format (KB, MB, GB). Shows used, available, and total space.
