> Console Log with Variable Name
```javascript
const myVariable = 'Hello World';
console.log({ myVariable });
```
**Function:** Logs both the variable name and value, making debugging easier than `console.log(myVariable)`.

> Array Destructuring
```javascript
const [first, second, ...rest] = ['a', 'b', 'c', 'd'];
```
**Function:** Extract values from arrays into distinct variables. The rest operator captures remaining elements.

> Template Literals
```javascript

```
**Function:** String interpolation using backticks and `${}` syntax for embedding expressions.

> Arrow Function
```javascript
const add = (a, b) => a + b;
const greet = name => `Hello ${name}`;
```
**Function:** Concise function syntax. Parentheses optional for single parameters, curly braces optional for single expressions.


> Ternary Operator
```javascript
const message = age >= 18 ? 'Adult' : 'Minor';
```
**Function:** A shorthand for simple if-else statements. Condition ? valueIfTrue : valueIfFalse.

  data-component="TestItOut"
  data-title="JS Snippets Playground"
  data-description="Try different ages and conditions! Experiment with nested ternary operators but keep them readable."
  data-snippet-id="js-common-0-0-9"

  const message = age >= 18 ? 'Adult' : 'Minor';
  console.log(`Age ${age}: ${message}`);
  &#47;&#47; Try with different ages
  const checkAge = (userAge) => {
    const status = userAge >= 18 ? 'Can vote' : 'Cannot vote yet';
    return `${userAge} years old: ${status}`;
  };\n
  console.log(checkAge(16));
  console.log(checkAge(21));
  console.log(checkAge(65));
  &#47;&#47; Nested ternary (be careful!)
  const score = 85;
  const grade = score >= 90 ? 'A' : score >= 80 ? 'B' : score >= 70 ? 'C' : 'F';
  console.log(`Score ${score}: Grade ${grade}`);">
</div>

> Default Parameters
