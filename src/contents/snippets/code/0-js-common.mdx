These snippets are incredibly common and are useful in everyday JavaScript programming. Some of these will become so familiar that you will not even think about them anymore, others it is just easy some time to cut and paste!

---

> Console Log with Variable Name
```javascript
const myVariable = 'Hello World';
console.log({ myVariable });
```
**Function:** Logs both the variable name and value, making debugging easier than `console.log(myVariable)`.

<br/>

---

<br/>

> Array Destructuring
```javascript
const [first, second, ...rest] = ['a', 'b', 'c', 'd'];
```
**Function:** Extract values from arrays into distinct variables. The rest operator captures remaining elements.

<br/>

---

<br/>

> Object Destructuring with Renaming
```javascript
const { name: userName, age: userAge } = user;
```
**Function:** Extract object properties and assign them to variables with different names.

<br/>

---

<br/>

> Template Literals
```javascript
const message = `Hello ${name}, you are ${age} years old`;
```
**Function:** String interpolation using backticks and `${}` syntax for embedding expressions.

<br/>

---

<br/>

> Arrow Function
```javascript
const add = (a, b) => a + b;
const greet = name => `Hello ${name}`;
```
**Function:** Concise function syntax. Parentheses optional for single parameters, curly braces optional for single expressions.

<br/>

---

<br/>

> Array Map
```javascript
const doubled = numbers.map(n => n * 2);
```
**Function:** Transform each element in an array and return a new array with the results.

<br/>

---

<br/>

> Array Filter
```javascript
const adults = users.filter(user => user.age >= 18);
```
**Function:** Create a new array with elements that pass a test condition.

<br/>

---

<br/>

> Find Element in Array
```javascript
const user = users.find(user => user.id === targetId);
```
**Function:** Returns the first element that matches the condition, or undefined if none found.

<br/>

---

<br/>

> Ternary Operator
```javascript
const message = age >= 18 ? 'Adult' : 'Minor';
```
**Function:** A shorthand for simple if-else statements. Condition ? valueIfTrue : valueIfFalse.

<br/>

---

<br/>

> Default Parameters
```javascript
function greet(name = 'Guest', age = 0) {
  return `Hello ${name}, age ${age}`;
}
```
**Function:** Provide default values for function parameters when no argument is passed.
