> Nullish Coalescing Operator
```javascript
const username = user.name ?? 'Anonymous';
```
**Function:** Returns the right side if left side is null or undefined (but not for falsy values like 0 or '').

console.log('undefined:', user5.name ?? 'Anonymous');\n
console.log('With ||:');
console.log('Alice:', user1.name || 'Anonymous');
console.log('null:', user2.name || 'Anonymous');
console.log('empty string:', user3.name || 'Anonymous');
console.log('zero:', user4.name || 'Anonymous');
console.log('undefined:', user5.name || 'Anonymous');">

> Object.entries() for Iteration
```javascript
Object.entries(obj).forEach(([key, value]) => {
  console.log(`${key}: ${value}`);
});
```

**Function:** Convert object to array of [key, value] pairs for easy iteration over object properties.


> Function Composition
```javascript
const pipe = (...fns) => value => fns.reduce((acc, fn) => fn(acc), value);
const process = pipe(trim, toLowerCase, capitalize);
```
**Function:** Combine multiple functions into a single function that applies them in sequence.

> Debounce Function
```javascript
const debounce = (func, delay) => {
  let timeoutId;
  return (...args) => {
    clearTimeout(timeoutId);
    timeoutId = setTimeout(() => func(...args), delay);
  };
};
```
**Function:** Limit function execution frequency, useful for search inputs or resize events.
